# hls-pr2mgrs

Landsat 8 PathRow to Sentinel 2 Tiles.

### Install

```
$ pip install pip -U
$ pip install .[deploy]
```

### Deploy 

```
$ cd stack
$ cdk init && cdk deploy
```


### Lambda Invocation
